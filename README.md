# Concurrent_Systems_Project
This project is simply the implementation of what is required in the assignement. The project was developed using the Java programming language and the FSP language for the design and implementation model.

## About 
This work has been realized in University of Pisa/Scuola Superiore Sant'Anna in the context of Concurrent Systems exam.

## Path and dependency
```
Concurrent_Systems_Project/
├──assignment
├──lts_model
├──report
└──src

=======
assignment 		                -> the assignment of the project
=======
lts_model 						-> lets model written using the FPS language
=======
report 		                    -> report of the project
=======
src 		     					 -> source code
=======
```
